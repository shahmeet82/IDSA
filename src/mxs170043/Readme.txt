Steps to run the program:-

This is a java program, so to compile this you will need to JDK installed on your system.
To compile this program type: javac filename.java from its destination.
After compilation is successful we can run the program using: java filename

Driver program:-

This program peforms the following actions on:-

Bounded Queue Operations:
1. Offer
2. Poll
3. Peek
4. Print Size
5. Clear Queue
6. Convert to Array
7. Print Queue
8. Is Empty?
